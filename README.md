# Project Titan V3

A monolithic Next.js ecommerce portfolio project demonstrating elite UI/UX design and full‑stack engineering.

## Setup

1. Clone or extract this folder.
2. Run `npm install`
3. Run `npm run dev` to start the development server.
4. Visit `http://localhost:3000`

## Features (Phase 1)

- **Design System** – `/design-system` showcasing all UI components.
- **Homepage** – Hero, categories, trending products, benefits, testimonials, newsletter, footer.
- **Responsive** – Mobile‑first, works on all devices.
- **Accessible** – Keyboard navigation, ARIA labels, semantic HTML.
- **Performant** – Optimized images, lazy loading, minimal bundle size.

## Technologies

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui (customized)
- Framer Motion
